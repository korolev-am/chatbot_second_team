# Чат-бот "Я всё сам"
 
 Схема работы чат бота:

![Схема](.git/diagram3.png)

Документация(на данный момент: [URL](http://localhost:9191/ws/docs)

Запуск python-приложения

```
make
```

Или

```bash
cd ./backend
pip3 install poetry
poetry install
poetry run start
```

Запуск svelte-приложения


```
make
```

Или

```bash
cd ./frontend
npm i
npm run dev
```
